Abstract · 摘要

本文针对无人机视觉跟踪任务面临的三类核心挑战—— 多模态融合的可信度建模缺失、 模板表征对目标遮挡与模态退化的敏感性、 小目标短暂消失场景下的重检测能力不足—— 以 SUTrack 单流跟踪框架为基础，分别在 架构 / 训练 / 推理 三个层面提出三个核心模块。

本文方法在 Anti-UAV、Anti-UAV410、DUT Anti-UAV 与 CST Anti-UAV 四个公开无人机基准上系统验证。在 Anti-UAV 红外协议上相比 UAUTrack 在 SA、AUC、P@20、PNorm 四个核心指标上分别提升 1.91, 4.20, 4.43, 4.29 个百分点； 在 CST 极小目标基准上 SA(0.30) 达到 39.18， 超越当前公开最优方法 GlobalTrack 共 3.96 个百分点，是已知公开方法中首个在该基准上将 状态准确率突破 39.0 的算法。

方法概述 · Method Overview

本文方法以 SUTrack 单流跟踪框架为基础，针对无人机视觉跟踪的三类核心挑战， 在架构 / 训练 / 推理三个层面分别引入三个互相独立又协同工作的核心模块：

架构层

VAAF

可见性自适应融合

Visibility-Aware Adaptive Fusion

  • 三头可见性预测（RGB / IR / 目标）
  • 可见性驱动的跨模态门控
  • 分组正交低秩适配（GOLA）
训练层

RMD

鲁棒模态蒸馏

Robust Modality Distillation

  • Cox 过程模板随机掩码
  • 跨模态内容-风格双蒸馏
  • 双向梯度耦合的协同训练
推理层

MGTR

记忆引导跟踪与重检测

Memory-Guided Tracking & Re-detection

  • 八维卡尔曼运动建模
  • 质量门控的回溯式记忆库
  • 运动感知重排序与重检测

演示视频 · 多模态实景跟踪

三段真实 Anti-UAV 测试序列上的本文完整方法运行结果。 每段视频左 = RGB 可见光 · 中 = IR 红外 · 右 = 实时可见性 HUD。
说明：Anti-UAV 数据集 RGB 与 IR 非像素对齐 (640×512 vs 1920×1080)， 模型仅在 IR 坐标系输出预测，RGB 黄色框为 GT 真值参考位置。

白天清晰场景 Day-clear

目标在白天清晰场景下被持续跟踪。两个模态都可靠，VAAF 给出接近平衡的融合权重。

黄昏 + 目标短暂消失 MGTR demo

黄昏序列含目标短暂消失。注意可见性时序图：v_target 跌至 0 时跟踪器主动输出 ABSENT， 目标重新出现后由 MGTR 记忆库 + 卡尔曼运动先验快速重定位。

夜间低光 (VAAF 救场) VAAF rescue

夜间低光场景。RGB 通道几乎全黑，目标信息完全来自 IR 红外通道。 VAAF 通过 v_rgb 自动降低对 RGB 路径的吸收，跨模态门控有效压制 RGB 噪声注入。

A/B 对比 · 模块价值演示

同一段序列上"完整方法"与"去掉某模块"的并排跟踪。 上半绿框 = 本文方法 · 下半红框 = 消融变体 · 右侧实时显示 IoU vs GT。

VAAF 价值演示 Night Low-light

夜间 RGB 全黑、目标信息只来自 IR。完整方法（绿）的 v_rgb 自动收敛接近 0、 跨模态门控关闭对 RGB 的吸收，IoU 维持 0.89；移除可见性头后（红，a04 变体） IR 路径仍接收 RGB 噪声，跟踪在帧 80 之后丢失目标，IoU 跌至 0。

MGTR 价值演示 Dusk + Absence

黄昏序列含 30 个目标缺失帧。完整方法（绿）通过目标存在性头与质量门控记忆库 识别缺失，主动输出 'absent / no output'；移除 MGTR 推理后（红）跟踪器在缺失帧上 仍输出错误位置的 bbox（如 frame 110 的红色大框），属 SA 评测协议下的虚警。

数据集 · Datasets

本文方法在 4 个公开无人机视觉跟踪基准上系统验证。

Anti-UAV RGB + IR

中国科学院 Jiang 等人发布的多模态无人机基准。 160 段训练 / 67 段验证 / 91 段测试。 每段视频含可见光与红外两路同步采集，覆盖白天、夜晚、阴雨等多种场景。

官方主页 · GitHub

Anti-UAV410 IR · 长序列

北京理工大学 Huang 等人发布的扩展红外基准。 410 段视频共约 438K 帧，平均序列长度超过 1000 帧， 对长时序跟踪与目标重检测能力提出更高要求。

下载 (官方仓库) · 论文 (arXiv)

DUT Anti-UAV RGB · 跨域

大连理工大学 Zhao 等人发布的纯可见光无人机数据集。 本文实验中作为零样本跨域评测集使用，反映模型对训练域外纯可见光场景的泛化能力。

GitHub

CST Anti-UAV IR · 极小目标

南昌航空大学 Xie 等人发布的复杂场景小目标基准。 220 段红外视频，目标平均像素面积低于图像总面积的万分之一， 是当前公开数据集中难度最高的红外无人机基准。

GitHub
